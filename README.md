# AG Water Softener

Static homepage/PDP for `agsoftener.com`.

## Local Preview

Serve the folder with any static file server, then open `/`:

```sh
python3 -m http.server 4174 --bind 127.0.0.1
```

Supported preview query params:

- `?band=hard`
- `?band=veryhard`

Checkout is intentionally disabled for this version. CTA buttons open a cart drawer with `Checkout opening soon`.
